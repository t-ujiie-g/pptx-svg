Interactive Editor

E6 editing features — how to test
  • Undo/Redo (E6.1): any edit, then the Undo/Redo buttons or Ctrl+Z / Ctrl+Y.
  • Inline text (E6.2): select a text shape → ✎ Inline Text → click in the text (boxes show per-char geometry), type, Backspace, Enter; Esc exits.
  • Z-order (E6.3): select a shape (overlapping ones help) → Front / ▲ / ▼ / Back.
  • Group move (E6.4): Shift+click several shapes → arrow keys move them together as one undo step.
  • Copy/Paste (E6.5): select a shape → Copy (Ctrl+C), change slide, Paste (Ctrl+V). Works with images.
  • Table (E6.6): select a table → set R/C + text and Set Cell, or +Row/−Row/+Col/−Col.
